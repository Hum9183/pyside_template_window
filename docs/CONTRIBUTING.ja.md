# コントリビューションガイド

Maya PySide テンプレートプロジェクトへの貢献をありがとうございます。

[English Version](../CONTRIBUTING.md)

## 開発環境

- **Maya**: 2022 以上
- **VS Code**: 推奨エディタ
- **自動フォーマット**: 保存時に Black + isort が自動実行

## 基本ルール

### ブランチとコミット
- 作業用ブランチを作成してから開発
- 1つの変更につき1つのコミット
- 分かりやすいコミットメッセージ

### コードスタイル
- **型アノテーション**: 関数・メソッドに必須
- **PySide対応**: `PySide6`→`PySide2` のフォールバック
- **ログ**: `print()` ではなく `logging` モジュール使用
- **文字列**: シングルクォート `'...'` が基本、Docstring は `"""`

## PR チェックリスト

提出前に確認：

- [ ] コードを自動フォーマット済み（Black + isort）
- [ ] 型アノテーションを追加済み
- [ ] Maya で動作確認済み
- [ ] PR説明にコード変更の意図を明記

## 注意事項

- **バージョン管理**: 管理者が実施（コントリビューターは対応不要）
- **ドキュメント更新**: 管理者が実施（コントリビューターは対応不要）
